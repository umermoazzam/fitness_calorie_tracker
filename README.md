## **Project Overview**

The Fitness Calorie Tracker is an interactive web application that helps users:

- Track daily **calorie intake**
- Estimate **calories burned** during activities
- Plan **weight gain or loss** with recommended foods and daily calorie targets

It‚Äôs user-friendly and includes visual dashboards for tracking progress.

---

## **Features**

1. **Login / Signup System**
   - Secure user login and registration
   - Personalized session per user

2. **Calorie Intake Tracker**
   - Search food items and input quantity
   - Calculate and log daily calorie intake

3. **Calories Burned Estimator**
   - Calculate calories burned based on weight and distance walked

4. **Daily & Weekly Progress Dashboard**
   - Pie chart for daily intake vs burned calories
   - Weekly bar chart of intake and burned calories

5. **Weight Gain / Loss Planner**
   - Calculate daily calorie surplus or deficit to reach target weight
   - Recommended high- or low-calorie foods
   - Tips for healthy eating and exercise

6. **Developer Info**
   - Display developer names in sidebar
